{{title}}

{{title}} provides placeholder images for every case, web or print, on almost any subject, in any size. It's simple and free. Just put the custom url in your code like so:

<img src="{{url}}/320/240" />

Photos come from Flickr and have a Creative Commons license.

Examples

{{url}}/320/240

A random picture of 320 x 240 pixels. If not supplying any keyword, you'll get a picture matching the keyword puppy.

{{url}}/320/240/cat

A random picture matching the keyword cat, of 320 x 240 pixels.

{{url}}/320/240/brazil,rio

A random picture matching the keywords brazil or rio, of 320 x 240 pixels.

{{url}}/320/240/paris,girl/all

A random picture matching the keywords paris and girl, of 320 x 240 pixels.

If no matching photos are found, a photo of this dog is returned.

Multiple images on the same page

Your browser might cache the images when you request the same URL multiple times on the same page. You can resolve this by adding a meaningless querystring to the URL. So, for example...

<img src="{{url}}/320/240?random=1" />
<img src="{{url}}/320/240?random=2" />
<img src="{{url}}/320/240?random=3" />
